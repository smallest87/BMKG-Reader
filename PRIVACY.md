# Kebijakan Privasi

Ekstensi **BMKG Weather Reader** dirancang dengan prinsip privasi:

1. **Tidak Ada Pelacakan:** Ekstensi ini tidak menggunakan Google Analytics atau pelacak pihak ketiga.
2. **Penyimpanan Lokal:** Semua data yang diambil (scrape) hanya berada di memori browser Anda (RAM) dan hilang saat Anda menutup panel atau menyegarkan halaman.
3. **Tidak Ada Pengiriman Data:** Ekstensi ini tidak mengirimkan data cuaca atau data pribadi Anda ke server mana pun.
4. **Izin Akses:** Izin `activeTab` hanya digunakan untuk membaca struktur HTML halaman BMKG saat tombol ditekan.
