## Cara Berkontribusi

1.  **Fork** repositori ini ke akun GitHub Anda.
2.  **Clone** repositori yang sudah di-fork ke komputer lokal Anda.
3.  Buat **Branch** baru untuk fitur/perbaikan Anda:
    ```bash
    git checkout -b fitur-baru-anda
    ```
4.  Lakukan perubahan kode. Pastikan kode rapi dan tidak merusak fitur yang sudah ada.
5.  **Commit** perubahan Anda dengan pesan yang jelas:
    ```bash
    git commit -m "Menambahkan fitur grafik curah hujan"
    ```
6.  **Push** ke branch Anda:
    ```bash
    git push origin fitur-baru-anda
    ```
7.  Buka **Pull Request (PR)** di repositori asli ini. Jelaskan apa yang Anda ubah.

## Pelaporan Bug
Jika menemukan error, silakan buat **Issue** baru dengan menyertakan:
-   Langkah-langkah untuk mereproduksi error.
-   Browser yang digunakan (misal: Chrome v120).
-   Screenshot error (jika ada).
